Listas



Las listas nos permiten crear conjuntos de elementos relacionados entre sí de alguna manera, por ejemplo un índice. Existen tres tipos de listas:

Listas ordenadas: muestra los elementos listados con letras o números.

Estructura:

<ol>...</ol>: delimitan la lista completa. <li>...</li>: delimitan cada uno de los elementos de la lista.

Ejemplo de estructura:

<ol>
<li>Elemento 1</li>
<li>Elemento 2</li>
<li>Elemento 3</li>
</o>

Ejemplo:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Listas desordenadas: muestra los elementos listados con puntos.

Estructura:

<ul>...</ul>: delimitan la lista completa. <li>...</li>: delimitan cada uno de los elementos de la lista.

Ejemplo de estructura:

<ul>
<li>Elemento 1</li>
<li>Elemento 2</li>
<li>Elemento 3</li>
</o>

Ejemplo:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Listas de definición: muestra los elementos sin identificador.

Estructura:

<dl>...</dl>: delimitan la lista completa. <dt>...</dt>: delimitan los terminos. <dd>...</dd>: delimitan las definiciones.

Ejemplo de estructura:

<dl>
<dt>Término</dt>
<dd>Definicion</dd>
</dl>

Ejemplo:

Término
In at dolor ex. Nulla velit nisl, dignissim quis lorem nec, semper iaculis nisl. Fusce leo ante, molestie sed ligula.
Phasellus aliquam dignissim urna ut lobortis. Praesent tempus, urna pulvinar fermentum rhoncus, lacus mi finibus elit, at scelerisque risus risus.
