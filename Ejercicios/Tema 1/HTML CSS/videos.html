Vídeos

Para incrustar un vídeo en HTML se utiliza la etiqueta <video>.

Atributos

src: La dirección del video que se va a insertar.
autoplay: Es un atributo booleano; si se aplica, el vídeo comenzará a reproducirse automáticamente tan pronto como sea posible.
loop: Un atributo booleano; si se aplica, el vídeo se reproducirá de nuevo automaticamente al finalizar.
poster: Un enlace que indica la portada del vídeo hasta que el usuario lo reproduzca. Si no se especifica, no se mostrará nada hasta que el primer cuadro esté disponible.

En caso de querer insertar un vídeo de una página de internet, como por ejemplo YouTube, utilizaremos la etiqueta <iframe>, añadiendo los atributos width, height y src. Sin embargo, no hay que escribir el código a mano: solo hay que darle click derecho al vídeo de YouTube que queramos insertar y elegir la opción "Copiar código de inserción".
