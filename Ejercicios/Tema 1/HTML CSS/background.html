Background

  • background-color: se especifican con: un nombre de color válido, un valor HEX o un valor RGB.
  • opacity: Puede tomar valores del 0 (transparente) al 1(opaco).
  • background-image: toma el valor de una imagen externa, utilizando url(linkImagen).
  • background-repeat: se utiliza en compañía de background-image. Puede tomar los valores repeat-x, repeat-y, repeat y no-repeat
  • background-position: puede tomar los valores top, center, bottom o bien porcentaje o tamaño en combinación con left, right, center, top, bottom
